[8품사]

1. 8품사 종류

1) 명사: 사람/사물 등 이름 * 명사자리: ➀ 관사+명사 ➁ 관사+형용사+명사 ➂ 관사+지시형용사+명사 ➃ 소유격+명사 ➄ 전치사+명사 ➅ 타동사+명사 ➆ 구동사+명사

2) 대명사

인칭대명사 ex) I, you, he, she, they, it ....
지시대명사 ex) this, that, these, those
부정대명사 ex) some, any, all, one, another, the other

3) 동사: 주어의 동작/상태를 나타냄

Be동사 ex) am, are, is ....
일반동사 자동사: 동사 뒤 목적어 필요없는 동사
타동사: 동사 뒤 목적어 필요한 동사
조동사 ex) some, any, all, one, another, the other

# 동사의 형태관련 문제유형

➀ 조동사+동사원형
➁ Be동사+현재분사(~ing)/과거분사(~ed)
➂ 조동사 have+과거분사(~ed)
➃ 자동사/타동사 구분
타동사 discuss about/ reach at/ attend to/ resemble like/
regret for/ answer to/ exceed to/ approve for... 		전치사를 쓰면 틀림
자동사 listen to/ object to/ wait for/ respond to/ account
for/ deal with/ talk about/ rafer to/ interfere with/
comply with/ depend on/ care for... 		전치사를 없으면 틀림

4) 형용사
역할: 명사 수식/ 주격보어자리/ 목적격보어자리
ex) I met some nice students. / They were cool. / The movie made me happy.

5) 부사: 동사/형용사/부사/문장 전체 수식 (명사는 수식 X)
ex) We can meet frequently. / The class was so easy to understand.
He speaks very fast. / Recently, I began to study English.

6) 전치사: 명사 앞에 놓여 장소, 방향, 시간, 이유 등을 나타냄.

7) 접속사: 단어-단어, 구-구, 절-절 연결

등위접속사 대등하게 연결 ex) and, but. or ...
상관접속사 짝을 이루어 쓰임 ex) both A and B, either A or B, neither A nor B,
not only A but also B, B as well as A ...
종속접속사 주절-종속절 연결 ex) because, although, since, before, after ...

8)감탄사: 감정 전달을 위해 입에서 내는 소리

2. 명사

1) 가산 명사: 셀 수 있는 명사
단수: 부정관사 or 정관사 / 복수: +(e)s

2) 불가산 명사: 셀 수 없는 명사 / 단수 복수 구분없이 항상 단수 취급

3) 한정사: 명사 앞에 놓여 명사의 의미를 한정

➀ 관사: 부정관사(a/an), 정관사(the)
➁ 소유격: 인칭대명사의 소유격 ex) my, yours, his...
➂ 지시형용사: this, that (these, those)
➃ 수량형용사: many, much, few, a few, little, a little, a lot of...

4) 복합명사: 두 개 이상의 명사가 한 단어처럼 쓰이는 경우
ex) account number, application form, insurance coverage, job appraisal,
safety procedure, customs office, savings account...

5) 일반명사-사람명사 구분

engineering / enginner
application / applicant
authorization / author
performance / perfomer
translation / translator
attendance / attendant
consultation / consultant
contribution / contributor
occupation / occupant

3. 대명사: 명사 반복 피하기 위해 대신해서 사용

1) 인칭대명사: 사람/사물 지칭하는 대명사

2) 재귀대명사: 문장의 주어/목적어 같은 명사 쓰일 때 대신 사용

➀ 재귀용법 ex) He hit himself. (생략 불가능)
➁ 강조용법 ex) I myself hit him (생략 가능)
➂ 관용적용법
by onself (단독) for oneself (독력) of oneself (자동) on own oneself (독립)

3) 지시대명사: 사람/사물 가리킴: this, that, these, those
지시대명사 특별용법

➀ 한 문장 안에서 앞에 나온 명사 다시 언급할 때, that/those만 사용
ex) the bed in my room is smaller than that your room.
➁ those who: ~한 사람들
ex) Those (who are) interested in the movie will go to the theater tonight.

4) 부정대명사: 정해지지 않은 것들을 가리킴

➀ each/every: 단수취급
➁ some/any
some: 긍정문, 의문문 권유/부탁
any: 의문문, 부정문, 조건문, 긍정문강조의미
ex) I have some money. / Do you want some milk? /
I don't have any money. / Do you have any money? /
Any one can do that. / if you any money. lend me some.
➂ no/more
no: 형용사 취급
none: 대명사 취급

4. 형용사: 명사: 명사 앞 or 뒤 수식 / 주격보어 or 목적격보어 자리

1) 형용사 한정용법: 명사 앞/뒤 수식
ex) This is a good book. / I found out something special on the book.

2) 형용사 서술용법: 주격/목적격보어 자리
ex) she is nice. / she made me happy.

3) 수량형용사

➀ many, few, a few, a number of + 가산 복수 명사
➁ much, little, a little, an amount of + 불가산 명사
➂ each, every + 단수명사
➃ a lot of, plenty of + 가산/불가산 명사

4) 혼동되는 형용사

beneficial(유익한) / beneficent(인정많은)
successful(성공의) / successive(연속의)
comparable(필적하는) / comparative(비교의)
comprehensible(알기쉬운) / comprehensive(포괄적인)
considerable(상당한) / considerate(사려깊은)
responsible(책임지는) / responsive(반응하는)
respective(각각의) / respectful(공손한)
sensitive(민감한) / sensible(분별있는)

5. 부사

1) 역할: 동사 수식/ 형용사 수식/ 부사 수식/ 준동사 수식/ 문장 전체 수식

2) already, still, yet
ex) I already bought this. (벌써) / I still don't know this name. (여전히)
Have you finished it yet?. (벌써) / My sister cat't ride a bike yet.(아직)

3) very, much, so, too
ex) The phone is very small. / He enjoyed the movie much.
He speaks so fast. / It's too far to go there. (너무 ~해서 ~할 수 없다)

4) 부사/형용사 형태 같은 경우
hard / fast / high / late / near / early ...
ex) We did some hard work. / We worked hard.
I was late for the class. / He prepared the meeting late at night.

5) 혼동하기 쉬운 부사들

late - lately (최근에) close - closely (자세히) high - highly (매우)
most - mostly (대개) near - nearly (거의) hard - hardly (거의 ~않다)
